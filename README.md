---

## 🎨 Características Destacadas

*   **Motor 3D de Partículas Nativas:** Un sistema de constelación matemática, estela física de chispas y un cubo criptográfico 3D proyectado por software puro sobre un único canvas.
*   **Consola Live Unix Logger (Terminal):** Al hacer hover sobre los proyectos, un motor inyecta concurrentemente trazas reales de compilación, scrapers y testing simulando terminales Unix.
*   **Cursor Dual Inteligente (LERP):** Un sistema cinemático de seguimiento inercial para el cursor del sistema que se adapta de forma responsive e interactiva.
*   **Panel Desplizable de Inspección:** Selector y visor modular para examinar el dominio y el código fuente de los proyectos clave sin recargar la página.
*   **Audio Context UI Synthesizer:** Síntesis y oscilación de ondas sonoras nativas en tiempo real mediante Web Audio API para ofrecer feedback sonoro cyberpunk al hacer clicks y hovers.
*   **Transiciones Elásticas y Morphing:** Scroll suave (Lenis), marquesinas cinéticas reactivas que se deforman lateralmente (skew) por velocidad, y cambio degradado de colores de orbes ambientales según la sección.

---

## 🛠️ Stack Tecnológico

El proyecto apuesta por el máximo rendimiento técnico reduciendo el impacto de red y eliminando intermediarios de compilación:

*   **Core:** HTML5 Semántico & ES6 Vanilla JavaScript.
*   **Estilos:** CSS3 nativo avanzado (variables reactivas, film grain, orbes blur y transformaciones 3D).
*   **Animaciones & Scroll:** 
    *   [GSAP Core v3.12.5](https://greensock.com/gsap/) (Animación elástica avanzada)
    *   [GSAP ScrollTrigger v3.12.5](https://greensock.com/scrolltrigger/) (Sincronización de scroll)
    *   [Lenis v1.0.42](https://github.com/studio-freight/lenis) (Motor de suavizado de inercia)
*   **Tipografías:** Syne, Space Grotesk e Inter (vía Google Fonts).

## 🚀 Despliegue Local y Producción

### Servir en Local
Puedes abrir el proyecto instantáneamente en local usando cualquier servidor estático simple. 

Por ejemplo, con Node.js corre en tu terminal:
```bash
npx servor . index.html 9999
```
Luego abre en tu navegador: `http://localhost:9999`

### Despliegue a Producción
La web está lista para producción. Puedes subirla de forma gratuita e instantánea en plataformas como:
